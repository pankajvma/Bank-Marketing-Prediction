Bank Marketing Prediction

This Web interface is designed to work as a medium between our ML Model and the user. The ML model is developed for customer targeting in a Bank Marketing Campaign. It takes several inputs from the user and tries to predict whether a Customer can be a potential subscriber of a term deposit that is being offered by the bank.

After receiving the input query/queries from the user this Web Interface transfer the query to the API in which our ML model is implemented. The API in return gives us the prediction which is then displayed on the interface.
